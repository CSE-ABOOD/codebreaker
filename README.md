# Codebreaker

## Prerequisites
- Python 3.8+
- pygame >= 2.1.0
- numpy >= 1.24.0

## Setup & Run
```bash
git clone https://github.com/your-username/codebreaker.git
cd codebreaker
python3 -m venv venv
# macOS/Linux
source venv/bin/activate
# Windows PowerShell
.\venv\Scripts\activate.bat
pip install pygame numpy
python codebreaker.py
